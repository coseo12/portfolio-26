# Claude Code 글로벌 설정

## 언어

- 모든 응답은 **한국어**로 작성한다.
- 코드 내 주석도 한국어로 작성한다.
- 단, 변수명/함수명/클래스명은 영어로 유지한다.

---

## 응답 스타일

- 핵심을 먼저 말하고, 필요한 경우에만 부연 설명을 추가한다.
- 불필요한 반복이나 과도한 친절 표현은 생략한다.
- 코드 변경 시 변경한 이유를 간략히 설명한다.

---

## 코딩 원칙

### 일반

- 기존 코드의 스타일과 컨벤션을 최대한 따른다.
- 불필요한 코드 변경은 하지 않는다. 요청된 부분만 수정한다.
- 새로운 라이브러리 도입 시 반드시 먼저 물어본다.

### 코드 품질

- 함수는 단일 책임 원칙을 따른다.
- 의미 있는 변수명과 함수명을 사용한다.
- 매직 넘버는 상수로 분리한다.

### 주석

- 복잡한 로직에는 "왜(why)" 중심의 주석을 단다.
- 당연한 코드에는 주석을 달지 않는다.

---

## 작업 방식

- 큰 변경 전에는 계획을 먼저 제시하고 승인을 받는다.
- 확실하지 않은 내용은 추측하지 않고 질문한다.
- 에러 발생 시 원인 분석 → 해결책 제시 순서로 진행한다.

---

## 금지 사항

- 요청하지 않은 리팩토링 금지
- 요청하지 않은 파일 삭제 금지
- 검증되지 않은 외부 패키지 무단 추가 금지
